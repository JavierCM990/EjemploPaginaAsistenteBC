# Ejemplo Página Asistente BC

Ejercicio de Business Central desarrollado en AL.

## Descripción
Página de asistente (NavigatePage) con configuración asistida para agregar 
entradas a una lista de tareas pendientes.

## Archivos
- **ToDoAssistedSetup.Page.al** - Página asistente con 3 pasos y navegación
- **AddToDoAssistedSetup.CodeUnit.al** - Codeunit que registra la configuración asistida
- **AssistedSetupGroup.EnumExt.al** - Extensión de enumeración con valor Tasks

## Configuración
- Runtime: 16.1
- Application: 27.0.0.0
